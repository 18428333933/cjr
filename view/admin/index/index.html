1.将git上的项目先git clone +码云上的项目库路径 git到本地

2.将文件修改后----1.右键打开git bash 

                             2.git status(检查是否有可提交的文件)

                             3.通过(cd+文件路径)进入需要上传的文件

                             4.git add .(注：别忘记后面的.，此操作是把文件都添加进来）

                             5.git commit  -m  "提交信息"  （注：“提交信息”里面换成你需要，如“first commit”）

                             6.git push -u origin master   （注：此操作目的是把本地仓库push到github上面，此步骤需要你输入帐号和密码）

3.将文件上传到服务器:1.进入项目目录/data/wwwroot/bxb

                                     2.git push origin master
